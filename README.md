# Firework Raylib Simulation

This game has over 10,000 procedurally generated fireworks.Firework sounds also procedurally generated.

Game also have 4 visual effects for fireworks.
1.Wind.
2.Flash.
3.Shaking
4.Spakles

And you can enable auto-mode so that the fireworks explode on their own!


Made with C++ and Raylib 5.5
